# Privileged Access Management Lab (On-Prem)

## Overview

This project demonstrates a Privileged Access Management (PAM) model using a Windows Server 2025 environment.

It focuses on securing administrative access and detecting suspicious activity.

## Features

* Separate admin and user accounts
* Just-in-time privilege elevation
* Active Directory hardening
* Attack simulation (brute force, unauthorized access)
* Event log monitoring and analysis

## Skills Demonstrated

* PAM design principles
* AD security hardening
* Threat detection
* PowerShell automation
* Security auditing

## Setup

1. Configure Active Directory
2. Create admin and user roles
3. Apply GPO security policies
4. Deploy monitoring scripts
5. Run attack simulations

## Demo Scenarios

* Temporary admin access for IT staff
* Detect multiple failed logins
* Identify unauthorized file access

## Future Enhancements

* Integrate SIEM tools
* Add automated alerting system
* Expand to multi-server environment
